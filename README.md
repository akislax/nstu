Данный набор программ реализует функции для вычисления различных параметров трех геометрических фигур:

Прямоугольник
Треугольник
Окружность
Прямоугольник выполнял Андрей Самойлов
Треугольник выполнял Максим Куликов
Окружность выполнял Максим Фортуна
Для каждой фигуры предусмотрен отдельный файл .cpp, позволяющий рассчитать основные характеристики по заданным размерам
Прямоугольник

Данная программа реализует вычисление основных параметров прямоугольника по заданным длинам сторон

Функциональность
Программа запрашивает у пользователя высоту и ширину прямоугольника, после чего рассчитывает и выводит:

периметр
площадь
длину диагонали

Входные данные

height — высота прямоугольника (положительное вещественное число)
width — ширина прямоугольника (положительное вещественное число)
Ввод осуществляется через стандартный поток ввода (cin)

Используемые функции
В программе определены три статические функции:

1 perimetr (float, float) вычисляет периметр, возвращает float
2 square (float, float) вычисляет площадь, возвращает float
3 diagonalLenght (float, float) вычисляет длину диагонали, возвращает double, использует библиотеку cmath для sqrt и pow

Примечания

1 Программа предполагает корректный ввод чисел При вводе нечисловых значений поведение не определено
2 Для вычисления диагонали используется тип double, чтобы избежать потери точности при возведении в квадрат и извлечении корня
3 Код написан на C++ и использует стандартную библиотеку (iostream, cmath)


Треугольник

Данная программа реализует вычисление параметров треугольника по трем заданным сторонам

Функциональность
Программа запрашивает у пользователя длины трех сторон, проверяет условие существования треугольника, после чего рассчитывает и выводит:

Периметр
Площадь по формуле Герона
Результат проверки на равнобедренность (наличие хотя бы двух равных сторон)

Входные данные

a — первая сторона треугольника (целое число)
b — вторая сторона треугольника (целое число)
c — третья сторона треугольника (целое число)

Ввод осуществляется через стандартный поток ввода (cin)

Примечания

1 Программа включает логическую проверку: сумма любых двух сторон должна быть строго больше третьей Если условие не выполняется, выводится сообщение о невозможности существования фигуры
2 Для расчета площади используется функция sqrt из библиотеки cmath, так как она заменяет математический знак корня, недопустимый в синтаксисе C++
3 Переменная herons_area имеет тип double, чтобы корректно отображать дробную часть результата вычислений
