    Прямоугольник

Данная программа реализует вычисление основных параметров прямоугольника по заданным длинам сторон.

Функциональность

Программа запрашивает у пользователя высоту и ширину прямоугольника, после чего рассчитывает и выводит:

периметр площадь длину диагонали

Входные данные

height – высота прямоугольника (положительное вещественное число); width – ширина прямоугольника (положительное вещественное число). Ввод осуществляется через стандартный поток ввода (cin).

Используемые функции

В программе определены три статические функции:

perimetr (float, float) вычисляет периметр, возвращает float
square (float, float) вычисляет площадь, возвращает float
diagonalLenght (float, float) вычисляет длину диагонали, возвращает double, использует библиотеку cmath для sqrt и pow

Примечания

Программа предполагает корректный ввод чисел. При вводе нечисловых значений поведение не определено. Для вычисления диагонали используется тип double, чтобы избежать потери точности при возведении в квадрат и извлечении корня. Код написан на C++ и использует стандартную библиотеку (iostream, cmath).

    Треугольник

Данная программа реализует вычисление основных параметров треугольника по заданным длинам трёх его сторон. Программа запрашивает у пользователя стороны a, b и c, после чего проверяет условие существования треугольника (сумма двух сторон должна быть больше третьей). Если треугольник существует, программа рассчитывает и выводит: периметр, площадь по формуле Герона, а также результат проверки на равнобедренность. Входные данные: a, b, c – стороны треугольника (целые числа). Ввод осуществляется через стандартный поток ввода (cin).

В программе реализованы следующие алгоритмы: вычисление периметра как суммы всех сторон; нахождение площади по формуле Герона с использованием полупериметра и функции sqrt; проверка на равнобедренность через сравнение пар сторон на равенство. Программа написана на C++ с использованием библиотек iostream и cmath. В случае, если из заданных сторон невозможно составить треугольник, программа выводит соответствующее уведомление.


# Окружность

**Выполнил:** Максим Фортуна

Данная программа реализует вычисление основных параметров окружности по заданному радиусу.

## Функциональность

Программа запрашивает у пользователя радиус окружности, после чего рассчитывает и выводит:

- диаметр
- длину окружности
- площадь круга

## Входные данные

`radius` – радиус окружности (положительное вещественное число).  
Ввод осуществляется через стандартный поток ввода (`cin`).

## Используемые функции

В программе определены три статические функции:

- `diameter(double)` – вычисляет диаметр окружности, возвращает `double`
- `circumference(double)` – вычисляет длину окружности (периметр), возвращает `double`, использует математическую константу π
- `area(double)` – вычисляет площадь круга, возвращает `double`, использует математическую константу π
