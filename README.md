Данный набор программ реализует функции для вычисления различных параметров трёх геометрических фигур:

Прямоугольник
Треугольник
Окружность
Для каждой фигуры предусмотрен отдельный файл .cpp, позволяющий рассчитать основные характеристики по заданным размерам.

1. Прямоугольник

Данная программа реализует вычисление основных параметров прямоугольника по заданным длинам сторон.

Функциональность

Программа запрашивает у пользователя высоту и ширину прямоугольника, после чего рассчитывает и выводит:

периметр
площадь
длину диагонали

Входные данные

height – высота прямоугольника (положительное вещественное число);
width – ширина прямоугольника (положительное вещественное число).
Ввод осуществляется через стандартный поток ввода (cin).	
 
Используемые функции

В программе определены три статические функции:

1. perimetr (float, float) вычисляет периметр, возвращает float
2. square (float, float) вычисляет площадь, возвращает float
3. diagonalLenght (float, float) вычисляет длину диагонали, возвращает double, использует библиотеку cmath для sqrt и pow

Примечания

Программа предполагает корректный ввод чисел. При вводе нечисловых значений поведение не определено.
Для вычисления диагонали используется тип double, чтобы избежать потери точности при возведении в квадрат и извлечении корня.
Код написан на C++ и использует стандартную библиотеку (iostream, cmath).
